기본문법

1. 변수와 자료형

1) 변수 선언

- 함수 내부에서 'var 변수명;'으로 선언하면 함수에서만 사용 가능한 지역변수가 된다.
- 함수 내부에서 '변수명;'으로 선언하거나 함수 밖에서 'var 변수명;'으로 선언하면 전역변수가 된다.
- 전역변수와 지역변수가 동일한 이름인 경우, 함수 내부에서 호출하면 지역변수가 우선권을 가진다.
- 전역변수 사용 시 'window.변수명' 혹은 'this.변수명'으로 표현해서 사용한다. (window, this)
- 전역변수는 해당 window 내에서 어디든 사용할 수 있고, 지역변수는 해당 함수 내에서만 사용이 가능하다.

*참고 - window 객체는 브라우저에서 제공하는 브라우저창(window)에 대한 정보를 담고 있는 객체이다.

2) 자료형

자바스크립트에서는 변수에 별도로 자료형을 지정해 주지 않고 리터럴에 의해 자료형이 결정된다.

  • string(문자열)
  • number(숫자)
  • boolean(논리값)
  • object(객체)
  • function(함수)
  • undefined(초기화가 안된 변수)

2. 데이터 형변환

1) 문자열과 숫자의 '+' 연산

문자열 + 숫자 = 문자열
숫자 + 숫자 + 문자열 = 문자열 (숫자가 먼저 계산되고 뒤의 문자열과 합쳐진다.)

2) 강제 형변환

자바스크립트에서 문자형 -> 숫자형 변환 시
Number 내장 객체 또는 parseInt(), parseFloat() 내장 함수를 이용하여 강제 형변환이 가능하다.

자바스크립트에서 숫자형 -> 문자형 변환 시
String 내장 객체를 이용하여 강제 형변환이 가능하다.

3. 연산자

'==' 연산자와 '===' 연산자

'=='은 값이 일치하는지 확인하는 연산자이다.
'==='은 값과 자료형이 일치하는지를 확인하는 연산자이다.
